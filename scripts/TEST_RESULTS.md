## 실패 원인 분석

### 1. 영문 전용 상품명 (7건)
상품명이 영어로만 되어 있어 한글 키워드 매칭 실패:
- `Vitamin A 10000 고함량 180정 대용량`
- `Vitamin B-Complex 100 에너지부스트 180정`
- `Vitamin C 1000 타임릴리즈 대용량 240정`
- `Vitamin D3 10000 고함량 대용량 240소프트젤`
- `Omega-3 2000mg 초고함량 rTG 대용량 240캡슐`
- `Lutein 40mg 초고함량 눈건강 대용량 180소프트젤`

**해결방법**: DIRECT_CODE_MAP에 영문 키워드 추가 또는 영문-한글 매핑 추가

### 2. 숫자와 결합된 복합어 (4건)
비타민 이름과 숫자가 붙어있어 토큰 분리 실패:
- `솔가 비타민D3 5000IU 100소프트젤`
- `비타민D3 25mcg 1000IU 면역력 90캡슐`
- `닥터스베스트 비타민D3 5000 180소프트젤`

**토큰 결과**: `비타민d3` → DIRECT_CODE_MAP에 `비타민d`는 있지만 `비타민d3`는 없음

**해결방법**: 
1. DIRECT_CODE_MAP에 `비타민d3` 키워드 추가
2. 또는 토큰 정규화 로직에서 숫자 제거 (d3 → d)

### 3. 한글 복합어 미분리 (5건)
띄어쓰기 없는 한글 복합어:
- `롤화장지` (롤 + 화장지)
- `루테인지아잔틴` (루테인 + 지아잔틴)
- `비타민B컴플렉스` (비타민B + 컴플렉스)
- `오메가369` (오메가3 + 6 + 9)

**해결방법**: 한글 복합어 분해 로직 추가 또는 SYNONYM_MAP에 추가

### 4. 단수형/복수형 불일치 (4건)
`프로바이오틱` vs `프로바이오틱스`:
- `솔가 프로바이오틱 400억 30베지캡슐`
- `나우푸드 프로바이오틱-10 250억 50베지캡`
- `[라이프익스텐션] 프로바이오틱 Florassist 30억 30캡슐`
- `닥터스베스트 프로바이오틱 200억 30베지캡`

**해결방법**: SYNONYM_MAP에 `프로바이오틱` → `프로바이오틱스` 매핑 추가

## 개선 제안

### 즉시 적용 가능
1. DIRECT_CODE_MAP에 영문 키워드 추가:
   - `vitamin`, `omega`, `lutein` 등
   
2. SYNONYM_MAP 확장:
   ```javascript
   '프로바이오틱': ['프로바이오틱', '프로바이오틱스', '유산균'],
   '휴지': ['휴지', '화장지', '두루마리', '롤화장지'],
   ```

3. DIRECT_CODE_MAP에 숫자 결합형 추가:
   ```javascript
   '비타민d3': { code: '58910', path: '...' },
   '오메가369': { code: '73134', path: '...' },
   ```

### 장기 개선
1. 영문-한글 자동 변환 로직
2. 숫자 제거 정규화 로직 (선택적)
3. 한글 복합어 분해 알고리즘

## 결론

현재 DIRECT_CODE_MAP 기반 매칭 로직은 **80%의 정확도**로 잘 동작하고 있습니다.

실패한 20%는 대부분:
- 영문 전용 상품명
- 숫자가 결합된 변형 (D3, 369 등)
- 띄어쓰기 없는 복합어

이들은 간단한 SYNONYM_MAP 확장이나 DIRECT_CODE_MAP 키워드 추가로 해결 가능합니다.
